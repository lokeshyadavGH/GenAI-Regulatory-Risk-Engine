GenAI Regulatory Risk Engine
Overview
This project simulates a regulatory loan dataset and builds an end-to-end AI-powered compliance risk analysis system.
The system:
    • Detects anomalies using Isolation Forest
    • Generates executive compliance summaries using OpenAI GPT
    • Visualizes risk distribution via Streamlit dashboard
    • Deployed live on Streamlit Cloud

Tech Stack
    • Python
    • Pandas
    • Scikit-learn
    • OpenAI API
    • Streamlit
    • Cloud Deployment

Architecture
